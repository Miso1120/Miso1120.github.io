## React 개념

서로 연관된 코드들을 그룹핑 하고 이름을 붙이는 것

복잡한 내용은 숨기고 단순한 태그로 만드는 것

```
<Header><Header/> = <header>
						<h1>
							<a href="/">web</a>
						</h1>



<NAV></NAV> = <nav>
				<ol>
					<li><a href="/read/1">html</a></li>
					<li><a href="/read/2">css</a></li>
					<li><a href="/read/3">javascript</a></li>
				</ol>
			  </nav>
```

핵심은 "사용자 정의 태그를 만드는 것"
-> 이것이 부품으로 사용된다
=> 무궁무진한 활용 가능성

npm start
