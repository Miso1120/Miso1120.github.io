h1 태그 사용시 자동 줄바꿈
why? 제목 태그는 화면 전체를 사용하는 것이 편하기 때문
링크에 이런 기능이 적용된다면 불편할 것이기에 자신의 컨텐트 크기만큼만 사용한다.
### 중복의 제거

```
<style>


  h1, a{

   border-width: 5px;

   border-color: red;

   border-style: solid;

   border: 5px solid red;

   display: inline-table; /*인라인으로 사용한다. 즉 display의 기본값은 언제든 수정 가능하다.*/

  }

 </style>
```
